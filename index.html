Unsupervised Learning for Whisky Taste Segmentation

This project uses unsupervised learning (K-Means) to group Scottish single malt whiskies based on their flavor profiles like smokiness, fruitiness, and floral notes. By analyzing these clusters, the project reveals distinct taste groups linked to whisky regions such as Islay and the Highlands. The results include clear visualizations and provide a foundation for a basic whisky recommendation system. This shows how machine learning can help personalize whisky choices for both fans and sellers.
